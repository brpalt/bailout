# Game Documentation

## Overview
This game is an immersive experience where players navigate through challenges and engage with various characters.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Deployment:** Heroku

## Setup Instructions
1. Clone the repository: `git clone https://github.com/brpalt/bailout.git`
2. Navigate to the project directory: `cd bailout`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Gameplay Mechanics
- Players can move using arrow keys.
- Collect items to score points.
- Complete levels to advance.

## Character Classes
- **Warrior:** High health, melee damage.
- **Mage:** Low health, high spell damage.
- **Rogue:** Medium health, agile.
